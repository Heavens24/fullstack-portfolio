Introduction

Python is a high-level, interpreted programming language.

It is often used for web development, scientific computing, and data analysis.

print("Hello, World!")
  • Easy to learn
  • High-level language
  • Interpreted language
Variables and Data Structures

Variables are used to store values in Python.

Data structures include lists, tuples, and dictionaries.

x = 5 y = [1, 2, 3]
  • Lists are ordered collections of items
  • Tuples are ordered, immutable collections of items
  • Dictionaries are unordered collections of key-value pairs
Control Structures

Control structures are used to control the flow of a program.

They include if-else statements and loops.

if x > 5: print("x is greater than 5") for i in range(5): print(i)
  • If-else statements are used to make decisions
  • Loops are used to repeat actions
Functions

Functions are blocks of code that can be called multiple times.

They are used to organize code and reduce repetition.

def greet(name): print("Hello, " + name) greet("John")
  • Functions are defined with the def keyword
  • Functions can take arguments
Modules

Modules are files that contain Python code.

They are used to organize code and make it reusable.

import math print(math.pi)
  • Modules are imported with the import keyword
  • Modules can be used to access functions and variables
